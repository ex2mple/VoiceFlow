# VoiceFlow

Локальный аналог [Wispr Flow](https://wisprflow.ai) для macOS: зажми клавишу,
скажи — текст появится в активном приложении, уже почищенный ИИ. Всё работает
на устройстве, без интернета и подписок.

```
[Правый ⌥] → [Запись] → [Whisper: речь→текст] → [Ollama: чистка] → [⌘V в активное приложение]
```

- **Распознавание** — OpenAI Whisper (`large-v3-turbo`, квантованная) через
  whisper.cpp с Metal. Русский, английский и их смесь.
- **ИИ-чистка** — локальная LLM через Ollama (`gemma3:4b`): убирает слова-паразиты
  («э-э», «ну», «короче»), самоисправления («в пятницу… нет, в субботу» → «в субботу»),
  расставляет пунктуацию. Если Ollama не запущена — вставляется сырой текст,
  диктовка работает всегда.
- **Вставка** — через буфер обмена + синтетический ⌘V; прежнее содержимое
  буфера восстанавливается. Последние 10 диктовок — в меню.

## Сборка (без Xcode, только Command Line Tools)

```bash
make whisper   # один раз: собирает whisper.cpp (нужен cmake: brew install cmake)
make model     # один раз: качает модель Whisper (~550 МБ)
make ai        # один раз: ставит/запускает Ollama и качает gemma3:4b (~3.3 ГБ)
make cert      # один раз: локальный сертификат, чтобы разрешения переживали пересборку
make run       # собрать и запустить
make install   # положить в /Applications
make test      # юнит-тесты (свой раннер, XCTest недоступен без Xcode)
```

E2E-тест (синтезирует речь голосом Milena, гонит через настоящие Whisper и Ollama):

```bash
say -v Milena -o /tmp/d.aiff "Ну, короче, запиши меня на субботу" && \
afconvert -f WAVE -d LEI16@16000 -c 1 /tmp/d.aiff /tmp/d.wav
VOICEFLOW_E2E=1 VOICEFLOW_E2E_WAV=/tmp/d.wav make test
```

## Первый запуск

1. Приложение попросит **два разрешения**: Микрофон и Accessibility
   (для глобального хоткея и вставки текста).
2. Если модель Whisper не скачана — скачает сама, прогресс виден в menu bar.
3. Зажми **правый ⌥ Option**, говори, отпусти. Клавишу можно сменить в меню.

## Устройство

- `Sources/VoiceFlowCore` — логика без AppKit, покрыта тестами: санитайзер
  галлюцинаций Whisper, гейт тишины, валидатор ответа LLM (защита от того, что
  модель «ответит» на диктовку вместо чистки), клиент Ollama, история.
- `Sources/VoiceFlow` — приложение: хоткей (NSEvent global monitor),
  запись (AVAudioEngine → 16 кГц моно), вставка (⌘V через CGEvent), menu bar.
- `Sources/CWhisper` — заголовки whisper.cpp для Swift; статические библиотеки
  собирает `scripts/build-whisper.sh` в `vendor/whisper/lib`.
